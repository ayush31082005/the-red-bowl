# 🍜 The Red Bowl — v2.0 (Swiggy-Style)

Complete React + Tailwind frontend for The Red Bowl restaurant, Meerut.

## 🚀 Setup

```bash
npm install
npm run dev
# → http://localhost:5173
```

## ✨ What's New in v2

- **Swiggy-inspired design** — orange/white palette, clean cards, image-rich layouts
- **Real food images** via Unsplash CDN (no downloads needed)
- **36 menu items** across 8 categories with full English descriptions
- **3-slide auto-rotating hero** with crossfade transitions
- **Image-based category filter** (Swiggy-style circular filters)
- **Grouped menu** by category on the Menu page
- **Offers/deals banner** with gradient cards
- **Category showcase** grid with hover effects
- **Cart with discount logic** (₹50 off on orders above ₹500, free delivery above ₹350)
- **WhatsApp order** with full formatted cart message
- **8 customer reviews** with rating breakdown
- **Chef team section** on About page
- **Fully responsive** — mobile, tablet, desktop

## 📁 Structure

```
src/
├── components/   Navbar, Footer, MenuCard, CartDrawer, CategoryFilter, ReviewCard, WhatsAppButton
├── pages/        Home, Menu, About, Feedback, Contact
├── sections/     Hero, Offers, CategoryShowcase, FeaturedDishes, HowItWorks, Loyalty, Testimonials, CTA
├── context/      CartContext
├── data/         36 menu items + 8 reviews
└── utils/        formatPrice
```
